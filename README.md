# AngryBird Avoidance Game
AngryBird avoidance web game using your keyboard directions

## Tools
Visual Studio Code, Node.js

## Setting Modules
~~~
> npm init -y
> npm install express, ejs
> npm install nodemon --save-dev
~~~

## Game Start
`> tsc & nodemon index.js`
